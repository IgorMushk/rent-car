## Getting Started

1. Clone the repository:
```
git clone https://github.com/IgorMushk/rent-car.git
```
2. Navigate to the project directory:
```
cd rent-car
```
3. Install dependencies:
```
npm install
```
4. Run the application:
```
npm start
```
